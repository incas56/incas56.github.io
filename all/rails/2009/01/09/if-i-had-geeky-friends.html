If I had Geeky Friends

01 Jan 2009

I found a terribly minor bug in the datamapper gemspec file that was causing me an issue. I got on IRC and the guys told me to fork, fix, and send a pull request. I did and then my commit was pushed to github and my picture was on the sam/dm-core main page because I was the last “author.” It was like a brush with fame. I told #datamapper that I wished I had geeky friends so I could tell them.

My commit is still immortalized here. :)
